Q 1.7 Describe which are the appropriate filetypes for websites

The most common types of files are CSS, PNG, HTML, JPG/JPEG. HTML – short for Hyper Text Mark-up Language, represents the skeleton of a webpage and it includes everything from content to particular tags in the structure of the HTML file and it explains specific divisions on the webpage, from division of classes to where paragraphs start, end, line breaks and to specific styling of text in italic or bold. A similar file or document ends in .html. CSS – short for Cascading Style Sheets, used together with HTML they create the content of webpages to display the information and present it in a pleasing way, A CSS file can be used as a separate file from a HTML file in the scope of being more organized. The CSS feature permits software developers to create more interesting pages by adding and editing fonts, their sizes, background images, etc PNG/JPEG – are the most common names used for files. Contrasting PNG files JPEG files are more lossy, therefore each time we open and save a new photo it reduces it’s quality. MP3/MP4 – are files that are mostly used to contain videos and audios. Some other popular files are also GIFS & SVG.
